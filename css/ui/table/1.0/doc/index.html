table : 表格
王涛

示例

介绍

表格一共有3种尺寸，分别为行高为：22px、26px、30px；一共有3种样式

相关样式

样式名 状态 说明
.fui-table 必须 表格基础样式，为26像素行高尺寸，默认文案等inline节点水平居中
.fui-table-s 可选附加 配合.fui-table一起使用，使行高变为22像素
.fui-table-l 可选附加 配合.fui-table一起使用，使行高变为30像素
.fui-table-hover 可选附加 配合.fui-table一起使用，让行有hover状态背景色（需要:hover样式伪类支持）
.fui-table-striped 可选附加 配合.fui-table一起使用，使表格没有左右边框
.fui-table-action 可选附加 配合.fui-table一起使用，使表格变为图文大尺寸列表结构
.fui-table-align-right 可选附加 配合.fui-table一起使用，使表格内的inline元素默认居右
.fui-table-align-left 可选附加 配合.fui-table一起使用，使表格内的inline元素默认居左
.fui-a-l 可选附加 在单元格上使用，使子节点居左
.fui-a-c 可选附加 在单元格上使用，使子节点居中
.fui-a-r 可选附加 在单元格上使用，使子节点居右
.fui-odd 可选附加 在行上使用，使当前行背景色变为淡灰色
.fui-img 可选附加 在.fui-table-action单元格中使用，定义该单元格存放img标签
.fui-txt 可选附加 在.fui-table-action单元格中使用，定义该单元格存放文案
.fui-act 可选附加 在.fui-table-action单元格中使用，定义该单元格存放按钮或执行功能单元

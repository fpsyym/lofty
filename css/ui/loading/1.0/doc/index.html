介绍

加载状态浮出和遮罩功能依赖 fdev 的ui-dialog组件，功能配置请详见该组件文档
该dpl一共有两种尺寸，分别对应loading动画16像素、24像素。你可以修改其中的加载文案，请放心他们是自动水平居中的，无须在意字数限制等问题

相关样式

样式名 状态 说明
.fui-loading 必须 加载状态基础样式，loading动画为24像素
.fui-ico 必须 显示loading动画的容器
.fui-txt 必须 显示loading时提示文案的容器
.fui-loading-part 可选附加 配合.fui-loading使用，使loading动画变成16像素尺寸

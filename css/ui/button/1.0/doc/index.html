button : 按钮
钮也非

示例

介绍

按钮样式支持button和a标签两种写法，请在业务需求中根据视觉业务需要选择合适的方式。

相关样式

样式名称 状态 说明
.fui-btn 必须 按钮的基础样式，显示为灰色高26像素性状
.fui-btn-action 必须 该样式按钮主色为蓝色，对应一些功能确定状态
.fui-btn-important 必须 该样式按钮主色为橙色，对应重点突出按钮
.fui-btn-hint 必须 该样式按钮主色为黄色，对应提示按钮
.fui-btn-disabled 必须 该样式配合disabled="disabled"属性一起使用
.fui-btn-s 可选附加 使按钮变为22像素高度
.fui-btn-l 可选附加 使按钮变为32像素高度

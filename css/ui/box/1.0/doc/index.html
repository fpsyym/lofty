页面区块的容器的样式

相关样式

样式名 状态 说明
.fui-box 必须 模块的基础样式，每个模块必须含有此样式，显示为边框宽度为2像素的框体
.fui-box-gray 可选附加 和.fui-box一起使用，扩展的基础样式，使框体主体变为灰色
.fui-box-no-border 可选附加 和.fui-box一起使用，扩展的基础样式，显示为无框框体
.fui-box-s 可选附加 在基础样式的基础上使用此样式可将框体中的内边距变从16像素变为8像素
.fui-t 必须 标题框容器的样式
.fui-d 必须 内容框纵容器的样式
.fui-detail 必须 内容容器的样式
.fui-line 可选附加 和.fui-detail一起使用，可以为内容容器增加上边框

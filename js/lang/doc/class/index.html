Class : 类

介绍

面向对象类的实现。可以通过 class 快速创建一个新类，也可以快速的继承另一个类。

使用方法

创建一个新类： 
 
define(['lofty/lang/class'], function(Class) {
   var myClass = Class ({
	options: {
	  isAutoPlay: true,
	  countNum: {
		value:99,
	  }
	},
	init:function (config){
	  // do somthing
	}
  });
  return myClass;
});

创建一个继承自其它类的新类： 
 
// myClass将拥有 base 的所有方法，若有重名则覆盖。
define(['lofty/lang/class', 'lofty/lang/base'], function(Class, Base) {
   var myClass = Class ( Base, {
	options: {
	  isAutoPlay: true,
	  countNum: {
		value:99,
	  }
	},
	init:function (config){
	  // do somthing
	}
  });
  return myClass;
});

对于通过 Class 创建的类，若类中有 initialize 或者 init 函数，那么实例化类对象时，initialize 或 init 函数会被自动调用，通常我们选其一作为类的入口函数。

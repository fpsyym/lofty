log : 日志类
魏琪君

介绍

日志类，从 butterfly 迁移而来，高级浏览器下直接在控制台输出日志，IE 浏览器在可以把日志打印到页面上。

使用方法

define(['lofty/lang/log'], function(Log){
	var log = new Log('widget.Tabs');
	log.info('log info demo');
	log.warn('log warn demo');
	log.error('log error demo');
 });

配置方法

在 URL 中可以添加参数来查看不同等级、不同模块的日志： 
 &debug-log-level=info | error | warn | debug
 &debug-log-filter=...
 &debug-console=true | false

示例

http://sz99and1.cn.1688.com/?debug-log-level=info

也可以单独将某个模块的日志打印出来，需要在 URL 后增加类似 debug-log-filter=widget.Tabs 的参数

IE下示例

追加参数 debug-console=true 会在页面结尾渲染出日志内容，同时出现一个textarea可以允许我们在IE下运行js

http://sz99and1.1688.com/?debug-console=true&debug-log-level=info

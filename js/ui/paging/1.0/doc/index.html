分页组件

使用方法

 
define(['fui/paging/1.0'], function(Paging){
	//动态渲染（从空div使用JS创建分页组件）
	var paging = new Paging({
		container:'#dynamic_paging',
		onRender:function(index){
			console.log(index);
		}
	});

	//静态态渲染（利用页面上已有Dom创建分页组件）
	var paging = new Paging({
		tpl:"#static_paging"
	});
});

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
currentPage Number 1 当前页码：
totalPage Number 10 页码总数
listSize Number 9 页码出现省略号时，中间显示的页码个数
hasForm Boolean true 是否存在输入框以及按钮
showPrevNext Boolean true 是否显示“下一页”按钮
onRender Function NULL 点击页码时的回调函数
linkUrl String NULL 动态渲染分页时，若不传入onRender参数，则点击页码打开linkUrl，格式：http://xxx?page={page} 相对或绝对路径
classPrefix String 'fui' 默认的class前缀

公共方法

方法名称 入参说明 返回值 作用说明
render (page, totalPage) page:当前页码    totalPage:总页码 / 渲染页码函数

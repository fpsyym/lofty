placeholder : 输入提示
李世君

示例

介绍

在html5中有个placeholder属性，供可描述输入字段预期值的提示信息，该提示会在输入字段为空时显示，并会在字段获得焦点时消失。

placeholder 属性适用于以下的input类型：text, search, url, telephone, email 以及 password。当然也适用于textarea。

可惜的是IE9及其以下版本都不支持该属性，还有部分浏览器如Safari，Opera，IE10等尽管支持，但是展现形式又与firefox、chrome等常用浏览器不一致。

该组件就是为了这个问题而生，具体解决方案如下：

方案一：在支持placeholder的浏览器上，使用原生的placeholder， 不支持的浏览器使用该组件模拟。

方案二：因为有浏览器支持的形式不一致，为了统一，可以屏蔽浏览器默认属性placeholder，选择全部使用该组件来模拟。

使用方法

 
	define(['fui-placeholder'], function(placeholder){
		//初始化
		var pld = new placeholder({
			elemArr: [document.getElementsByTagName('input')[0], document.getElementsByTagName('textarea')[0]],
			shield: true
		});

		//全部对象重置位置
		pld.reset();

		//全部对象重置样式和位置
		pld.reset({
			color: 'red'
		})

		//部分对象重置样式和位置
		pld.reset({
			elemArr: [document.getElementsByTagName('input')[0]],
			color: 'red'
		});
   	});

注意点

elemArr里放置一个或多个需要使用该模拟功能的input或者textarea节点对象。

中间层中的提示语来源于input或者textarea中设置的placeholder属性值，字体颜色默认是#A9A9A9，可自定义。

中间层的样式采用input或者textarea中设置的样式，如：padding、line-height、font-family、font-size，垂直高度z-index值默认为2，可自定义。

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
elemArr Array [] 一个或多个input、textarea节点对象组成的数组
shield Boolean false shield设置为true，表示所有浏览器都使用该组件模拟；默认为false，只在不支持placeholder属性的浏览器中使用
color String #A9A9A9 color用来设置中间层中的提示语字体颜色
zIndex Number 2 zIndex用来设置中间层垂直高度
disableCls String disabled 触点禁用时的class
titleSelector String .fui-tab-t 触点元素集

公共方法

方法名称 入参说明 返回值 作用说明
reset() config 无 可以重新调整某些placeholder模拟层的样式和位置，默认重置所有模拟层的样式和位置

拖放组件

使用方法一：传入要被拖拽的节点元素的选择器（推荐）

define(['fui/dragdrop/1.0'], function(Dragdrop){
	//不设置拖放区域，任意拖放
	var dragdrop1 = new Dragdrop({
		targetSelector:'.targetSelector',
		bridgeSelector:'.bridgeSelector'
	});

	//设置拖放区域
	var dragdrop2 = new Dragdrop({
		targetSelector:'.targetSelector',
		bridgeSelector:'.bridgeSelector',
		area:[minX,maxX,minY,maxY]
	});
});

使用方法二：直接传入要被拖拽的节点元素（不推荐）

 
var dragdrop = new Dragdrop({
	target:Dom节点,
	bridgeSelector:'.bridgeSelector'
});

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
targetSelector String(targetSelector和target有且只能有一个) 无 被拖动的元素的选择器
target(不推荐使用) Dom节点(target和targetSelector有且只能有一个) 无 被拖放的元素节点
bridgeSelector String/可选 无 指定鼠标按下哪个元素时开始拖拽 (默认是按下整个需要被拖拽的元素)
dragable Boolean/可选 true 是否可以拖拽
dragX boolean/可选 true 是否可以沿X轴拖拽
dragY Boolean/可选 true 是否可以沿着Y轴拖拽
area Array/可选[minX,maxX,minY,maxY] 无 指定拖拽范围 (默认任意拖拽)
callback Function/可选 无 移动过程中的回调函数

公共方法

方法名称 入参说明 返回值 作用说明
dragX () / / 设置只能沿X轴拖拽
dragY () / / 设置只能沿Y轴拖放
dragAll () / / 设置任意方向拖放
setArea([minX,maxX,minY,maxY]) Array/[minX,maxX,minY,maxY] / 设置拖动范围
setDragable(b) boolean / 设置是否可以被拖动
getDragX() / boolean 获取是否可以沿着X轴拖动
getDragY() / boolean 获取是否可以沿着Y轴拖动

combobox : 选择下拉框
屈传鹏

示例

介绍

代替html默认的combobox组件

使用方法

 
<div id="test10" style="width:30%;"></div>
<select id="selectId" name="aaa" style="display:none;">
	<option value="111">111111111</option>
	<option value="222" selected="selected">222222222</option>
	<option value="333">333333333</option>
</select>

<script type="text/javascript">
	define(['fui/combobox/1.0'], function(Combobox){
		var combobox = new Combobox({
			container:"#test10",
			itemList:"#selectId"
		});
		
	});
	
</script>

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
readonly boolean false 文本框是否只读，默认可以编辑
disabled boolean false 整个组件是否禁用
name string "" 当组件是表单元素时，设置提交表单所用的name属性
itemList array | 自定义数据源 | 原生select元素的选择器字符串 空数组 下拉框选项的内容，数组格式为[{val:"xx",txt:"xx",selected:"selected"},{val:"xxx",txt:"xxx"}..]
height number：26 | 22 26 文本框的高度
showListener string："click" | "mouseenter" "click" 显示下拉框的触发事件
classPrefix string 'fui-combobox' class前缀

公共方法

方法名称 入参说明 返回值 作用说明
updateItemList(array) array:下拉框选项的数据源 组件实例本身 显示tip
addItem (index,o) index:要插入的位置，o:下拉框中一项 组件实例本身 在下拉框中指定位置增加一项，如果没有index参数，默认插入到最后
removeItem (index) index：要删除的项目的索引 组件实例本身 删除指定位置的选项
getItemList() / itemList 获取下拉框选项的列表
getText() / string 返回当前选择的选项的txt的值
getValue() / string 返回当前选择的选项的val的值
setIndex(index) number 组件实例本身 选中索引为index的选项
setReadonly(boolean) boolean 组件实例本身 改变文本框的只读属性
setDisabled(boolean) boolean 组件实例本身 改变组件的禁用状态

事件

事件名称 事件说明
change 当下拉框内容改变时触发该事件

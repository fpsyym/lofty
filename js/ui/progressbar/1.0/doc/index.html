progressbar : 进度条
钮也非

示例

介绍

进度条组件，如果展示方式不同，请覆盖tpl方法,并用css覆盖默认的css

使用方法

 
define(['fui/progressbar/1.0'], function(Progressbar){
	var progressbar = new Progressbar({
		container:'#progressbar',
		maxValue: 100
	});
});

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
maxValue Number 100 最大值
initValue Number 0 s初始值
enableAnim Boolean true enable动画
duration Number 2 动画持续时间
height Number 12 Processbar默认高度
create Function NULL progressbar初始时调用的回调函数
ready Function NULL 如果有动画，在动画结束后调用的回调函数
change Function NULL 当进度条的值改变时触发的回调函数
complete Function NULL 当进度条的值达到最大值时触发的回调函数

公共方法

方法名称 入参说明 返回值 作用说明
getValue () / Number 获得进度条当前数值
getPercentValue () / Number 获得进度条当前数值(百分比)
getMaxValue () / Number 获得最大数值
setValue(value) Number / 进度的绝对百分比。例如87%,则传入87
setPercentValue(percent) Number / 进度的百分比。例如87%,则传入87

标签切换组件

使用方法

 
//注意：若页面上有多个 Tabs 需要同时初始化，需要程序多次调用 new Tabs ({}) 来进行初始化。

define(['fui/tabs/1.0'], function(Tabs){
	//静态渲染Tabs组件（用页面上已有的Dom）
	var tab = new Tabs({
		tpl:'#staticTab'
	});

	//动态渲染Tabs组件
	var tab = new Tabs({
		container:'#dynamicTab',
		items: [{
			label: 'Tab1',
			content: '<p> Tab1 content</p> '
		}, {
			label: 'Tab2',
			content: '<p> Tab2 content</p> '
		}, {
			label: 'Tab3',
			content: '<p> Tab3 content</p> '
		}]
	});
});

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
autoPlay String: '' | 'line' | 'random' '' 是否自动进行切换 String 三个可选值：
interval Number 3000 自动切换的时间间隔 (单位：毫秒)
event String mouseenter 切换的事件触发类型
currentCls String fui-tab-current 当前tab的触点class
disableCls String disabled 触点禁用时的class
titleSelector String .fui-tab-t 触点元素集
boxSelector String .fui-tab-b 内容元素集
selected Number 0 设置初始化时第几个触点为当前状态
effect String: 'default', 'updown', 'leftright', 'fade' 'default' 内容元素集
duration Number 500 切换效果的默认时间
prev String .fui-tab-prev 上一页选择器
next String .fui-tab-next 下一页选择器
items Array [] 静态渲染时不需要设置该参数
动态渲染时需要设置为[{title:'tab1', content:'tab1 content'}, {}]的格式
classPrefix String 'fui' 默认的class前缀

公共方法

方法名称 入参说明 返回值 作用说明
switchTo (index, callback) index:标签的序列号    callback:回调函数 / 选中某个标签
prev () / Tabs对象自身 选中并显示上一个标签
next () / Tabs对象自身 选中并显示下一个标签
getLength() / Number 返回Tabs组件的标签个数
getCurrentIndex() / Number 返回选中标签的index

事件

事件名称 事件说明
beforeSwitch 选中标签发生变化前触发该事件
switch 选中标签发生变化后触发该事件

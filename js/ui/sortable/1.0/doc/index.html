sortable : 排序组件
王涛

示例

介绍

排序组件，给一组Dom元素添加排序功能，通过鼠标拖拽的方式来实现。

使用方法

 

define(['fui/sortable/1.0'], function(Sortable){
			
	var sort = new Sortable({
		tpl:'#demo1',
		items: '>li',
		placeholder: 'placeholder',
		cursorAt: {
			top: 5,
			left: 5
		},
		opacity: .5
	});
});

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
axis String: 'x' | 'y' false 如果被定义了，则选项只能被水平或者垂直托动,可选的值："x","y"
orientation String vertical 该配置的作用是辅助排序算法，指定模块的排序方向。当出现模块全部位于水平时需要使用"horizontal"
handle Boolean false 限制排序的动作只能在item元素的某个元素开始
items String '> *' 该配置为选择器，提供排序模块的节点条件
helper String 'original' 设置是否在拖拽元素时，显示一个辅助的元素。可选值： 'original','clone'
appendTo String 'parent' 指定helper追加在哪个元素后面
opacity Number 无 0 - 1 之间的数值，设置"helper"的透明度
placeholder String false 设置当排序动作发生时，空白占位符的CSS样式
dropOnEmpty Function | Boolean true 当"columns"有多个的情况下，出现某个"column"中没有"item"，是否允许从其他"column"拖拽"item"到空的"column"内
scroll Boolean true 拖拽过程中是否进行智能滚屏
scrollSensitivity Number 20 设置当元素移动至边缘多少像素时，便开始滚动页面
scrollSpeed Number 20 设置页面的滚动速度
revert Number 0 拖拽结束，释放鼠标后，"helper"到"placeholder"的一个动画效果的时长(毫秒)
zIndex Number 1000 拖拽时"helper"的"z-index"样式属性
classPrefix String 'fui' 默认的class前缀

事件

事件名称 事件说明
capture 当鼠标在"item"的"handle"区域点下(未释放)时，触发的事件。事件传递对象有：
 o:{
	currentItem: currentItem    //当前的"item"
  }
 e:event
start 当鼠标在"item"的"handle"区域点下(未释放)并开始移动时触发。事件传递对象有：
 o:{
    columns: self.columns,  //当前所有的column。假如未配置，则为初始化的容器本身
    items: self.items,  //当前的所有item
    currentItem: self.currentItem,  //当前被拖拽的item
    currentColumn: self.currentColumn,  //"placeholder"所在的column
    originalColumn: self.originalColumn,   //被拖拽的item在拖拽开始时所在的column 
    helper: self.helper,    //"helper"节点引用
    placeholder: self.placeholder   //"placeholder"节点引用
 }
 e:event
over 在拖拽过程中，切换column时触发。事件传递对象有：(同start事件)
change 在拖拽过程中，placeholder位置发生变化时触发。事件传递对象有：(同start事件)
stop 当拖拽结束并释放鼠标时触发。事件传递对象有：(同start事件)

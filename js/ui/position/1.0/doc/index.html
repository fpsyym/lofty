介绍

定位

使用方法

 
define(['fui/position/1.0', 'jquery'],function(Position, $){
	var position = new Position({
		targetNode:'.target',
		baseNode:'.base',
		offsetX:'left',             
		offsetY:'top'   
	});
});

使用说明

target元素css中的position属性为relative时，target元素相对base元素定位后，占位空间保留;非relative时，占位空间消失

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
winListen boolean:false|true false 被定位元素是否随窗口大小的变化而重新定位
targetNode String:jQuery选择器 无默认值，为必填项 需要定位的元素
baseNode String:jQuery选择器 'body' 定位参照元素
offsetX String：'left'|'right'|'center'|'50%'|'20px'|'-10'|'100%+20px'|'left+10px' 'left' targetNode相对于baseNode的水平方向的偏移
offsetY String:'top'|'bottom'|'center'|'50%'|'20px'|'-10'|'100%+20px'|'top+10px' 'top' targetNode相对于baseNode的垂直方向的偏移

公共方法

方法名称 入参说明 返回值 作用说明
setOffsetX() offsetX / targetNode的水平方向的偏移量的设置
setOffsetY offsetY / targetNode的垂直方向的偏移量的设置
locate() / / 设置targetNode的left和top属性

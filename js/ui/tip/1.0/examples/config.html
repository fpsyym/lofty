最基础的配置
不带圆角
有关闭按钮，配置关闭前的处理方法，当方法返回true时，关闭tip
设置tip提示框与目标元素的相对方位:
1:top left 2:top right 3:bottom left 4:bottom right 5:left top 6:left bottom 7:right top 8:right bottom
设置宽度300px
微调位置(向上10px,向右20px)
isAnim:设置是否在显示和隐藏的时候有动画效果，默认为true，animTime可以设定动画持续的时间，默认为50毫秒
isOnloadShow：true页面加载完成tip自动显示，onloadHold可配置显示的时间，不配置时间时默认一直显示（除非其他触发消失的事件被触发时关闭）

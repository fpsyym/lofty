tip : 提示框

介绍

提示框

使用方法

 
define(['fui/tip/1.0'], function(Tip){
	var tip = new Tip({
		target:"#test1",
		content:"message"
	});
});

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
target String:jQuery选择器 '' tip作用的目标元素
isRoundedCorner boolean true tip是否有圆角
hasCloseButton boolean false 是否有关闭按钮
onCloseButtonClick function 空方法 当有关闭按钮，并且点击关闭按钮时的回调方法，回调方法返回false的话会阻止tip的关闭
content String "" tip显示的内容
local number 1 tip提示框相对目标元素的方位： 1:top left 2:top right 3:bottom left 4:bottom right 5:left top 6:left bottom 7:right top 8:right bottom
arrow number 9 箭头相对tip提示框的方位：0:null 1:top left 2:top right 3:bottom left 4:bottom right 5:left top 6:left bottom 7:right top 8:right bottom 9:default
width Number 150 设置提示框的宽度
dLeft number 0 提示框向左微调的像素数，当为负数时表示向右微调
dTop Number 0 提示框向上微调的像素数，当为负数时表示向下微调
isAnim boolean true 提示框在显示和隐藏时是否有动画效果
animTime number 50 动画效果持续的时间，单位毫秒
isHoverHold boolean true 鼠标经过tip提示框时，提示框是否保持显示状态
hideDelay number 200 当需要隐藏tip时，设置隐藏的延迟时间，单位毫秒，默认为200毫秒，当设置为0时，不延迟
isOnloadShow boolean false 页面加载完成，是否自动显示tip
onloadHold number 一直显示 页面加载完成后显示tip的时间
overflowChange boolean true 当tip提示框溢出页面时，是否在目标元素的反方向显示（例：tip显示在目标元素上方，当溢出时，自动显示在目标元素的下方）
isAutoHide boolean true 是否自动隐藏，默认true（滚动条滚动，document其他元素点击时，tip会消失）
showListener string 'mouseenter' 显示tip的触发事件,当配置为空时，需要调用.show()方法来显示tip
hideListener string 'mouseleave' 隐藏tip的触发事件,当配置为空时，需要调用.hide()方法来隐藏tip
beforeShow function 空方法 设置在tip显示之前的自定义处理，当方法返回false时，将阻止tip框显示
classPrefix string 'fui-tip' 组件中class的默认前缀

公共方法

方法名称 入参说明 返回值 作用说明
show() / / 显示tip
hide () / / 隐藏tip
getCurrentTarget () / / 获取tip作用的当前的目标元素
setCurrentTarget(o) 当前的目标元素 / 由于jQury选择的元素是数组，此参数需要穿特定的元素，不能为数组。合法的例子：$("...")[0]或者document.getElementById("...")
setContent() string / 设置tip提示框的内容
setLocal() number / 设置tip提示框相对目标元素的方位
setArrow() number / 设置箭头相对提示框的方位
setWidth() number / 设置提示框的宽度
setDLeft() number / 设置提示框向左微调的像素数
setDTop() number / 设置提示框向上微调的像素数

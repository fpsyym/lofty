subcookie: 1688 网站前端专用 cookie
王涛

示例

介绍

subCookie是1688前端存放业务cookie的专用字段，原则上是不允许将前端cookie存放在其它cookie字段的！

对写cookie需要慎重，默认options是存在在1688.com域名下的，意味着你写的cookie会被客户带到网站的所有页面。故写操作时一定要限制子域及过期时间。

使用方法

 
 define(['alicn/subcookie/1.0'],function(SubCookie){
   SubCookie.get('hasTip');
   SubCookie.set('hasTip', true);
 })

公共方法

方法名称 入参说明 返回值 作用说明
get( String:key ) key: 需要获取的子 cookie 的 key String 读取子cookie
set ( String:key, String:value, Object:options ) key: 需要设置的子cookie的key
value: 需要设置的子cookie的value
options:
  • raw : Bolean 是否对value进行encode(escape/unescape)处理，默认false
 String 设置子cookie

介绍

地区选择组件

使用方法

 

define(['alicn/geoinfo/1.0', 'jquery'],function(Geoinfo, $){
				var geoinfo = new Geoinfo({
					container : $("div#container"),
					data: ["CN","浙江"]
				});
				geoinfo.on("change",function(data){
					var selectedArea = "";
					for(var nameIndex in data.name){
						selectedArea += data.name[nameIndex] + " ";
					}
					$('div.selected-area').html(selectedArea);
				})
			});

配置参数

配置参数 参数类型/可选值 参数默认值 作用说明
data Array | 地区数据预设 | 对于某些场景限制选择如“中国”下的地区，则使用该项

公共方法

方法名称 入参说明 返回值 作用说明
setData (data:Array) 地区预设数据 / 地区预设数据的另外一种设置方法，同直接配置
setDataSource (source:String) 数据源路径 / 对于某些场景，数据源不是默认的

事件

事件名称 事件说明
change 当下拉框内容改变时触发该事件，返回值带上所有选中的地区，以及当前是否最后一级

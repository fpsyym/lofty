aliuser: 阿里用户登录信息
王涛

示例

介绍

aliuser 组件是获取阿里巴巴网站用户登录状态及历史登录信息的组件，同时含有与淘宝会员打通的相关函数的支持。该组件是从 fdev4-min.js 中的 web.js 迁移而来，不再混入到框架的代码，而独立为 alicn 下的组件。

使用方法

 
define(['alicn/aliuser/1.0'],function(AliUser){
   var bLogin = AliUser.isLogin();		 //判断用户是否登录
   var loginId = AliUser.getLoginId()		 //提供当前登陆的用户Id，如果取不到则返回null
   var lastId  = AliUser.getLastLoginId();  //提供最近一次登陆过的用户Id，如果没有获取到则返回nul
})

公共方法

方法名称 入参说明 返回值 作用说明
isLogin ( ) / Boolean 判断用户是否登录
getLoginId ( ) / String 提供当前登陆的用户Id，如果取不到则返回null
getLastLoginId ( ) / String 提供最近一次登陆过的用户Id，如果没有获取到则返回null
getLastMemberId ( ) / String 取得上一次登录的memberId
updateLoginInfo ( ) {configs} deferred对象 从阿里集团的各个网站同步登录信息。
AliUser.updateLoginInfo({ source: ['b2b', 'taobao'] })

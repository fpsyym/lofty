misc : 公共 Util 函数

介绍

misc 是将一些比较常用而 jQuery 基础库中不存在的功能函数集合起来形成的类。Fdev4 的 gears.js 中的大部分 Util 函数可以在这个类中找到。

使用方法

				
  define(['util/misc/1.0']), function(Util){
	var html = "<script>"
	var encodeHtml = Util.escapeHTML(html); 
  });

公共方法

方法名称 入参说明 返回值 作用说明
substitute ( str, data ) str : {String} 字符串模板
data : {Object} 填充的数据 		返回字符串 提供简易的模板功能，例子： 
  Util.substitute('{0}+{1}={2}', [11, 24, 35]); //return "11+24=35";
  Util.substitute('{a} {b}!', {a:'I\'m', b:'google!'}); //return "I'm google!";
escapeHTML ( str, attr ) str : {Object} 需要转义的字符串
attr : {Bolean} 是否对属性进行额外处理 		转义后的字符串 将字符串进行HTML转义，使字符串能作为文本插入到HTML节点中显示
unescapeHTML ( str ) str : {Object} 需要转化的字符串 转义后的字符串 将字符串转化为HTML DOM结构
unparam ( s, separator ) s : {Object} 要hash的字符串
separator : {Object} 分隔符，默认 '&' 		转化后的对象 将字符串转换成hash, 例如：?key1=value1&key2=value2 ==> {key1:value1, key2:value2}
goTo ( url, target ) url : {String} 目标url
target : {String} 新开窗口or当前窗口 _self|_blank 		/ 新开窗口或者当前窗口打开(默认新开窗口),解决IE下referrer丢失的问题
isIE ( ) / true/false 判断当前浏览器是否为IE浏览器
isIE6 ( ) / true/false 判断当前浏览器是否为 IE6 浏览器
trim (str ) str : {String} 待处理字符串 处理后在字符串 去除字符串前后空白
lenB (str) str : {String} 待处理字符串 字符串长度 计算字符串的长度，该方法对中文及全角字符按两个英文字符计算
cut (str, len, ext ) str : {String} 待处理字符串 len : {Number} 需要截取的长度 ext : {String} 对截取后的字符串追加的字符串，默认不追加 返回截取后的新字符串 按照".lenB()"的规则计算长度并截取字符串，根据参数配置需要追加的字符串
